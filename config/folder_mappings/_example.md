# Folder Mapping — Example

> Create one file per account: `<account_id>.md`
> Use the account **nickname** as the heading for quick reference.
> The openclaw agent reads these for context when scoring emails (Step 3).
>
> The agent understands both the full account name and the nickname.

## [work] Work Account

| Folder | Description |
|--------|-------------|
| INBOX | Main inbox |
| Admin | IT support, helpdesk |
| Archive | Long-term storage |
| Executive | C-level / VIP communications |
| Finance | Invoices, billing |
| HR | Human resources, personnel |
| Newsletters | Marketing, newsletters |
| Projects | Project-related emails |
| Review | Needs manual triage |

## [personal] Personal Gmail

| Folder | Description |
|--------|-------------|
| INBOX | Main inbox |
| Archive | Storage |
| Finance | Banking, statements |
| Newsletters | Subscriptions |
| Review | Needs manual triage |
| Shopping | Orders, delivery |
| Travel | Bookings, itineraries |

---

*Copy this file to `<account_id>.md` and customize for your account.*
*Use the nickname in brackets `[nickname]` so the agent can reference it.*
