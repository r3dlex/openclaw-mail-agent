# Folder Mapping — Example

> Create one file per account: `<account_id>.md`
> Use the account **nickname** in brackets as the heading for quick reference.
> The openclaw agent reads these for context when scoring emails (Step 3).
>
> The agent understands both the full account name and the nickname.
